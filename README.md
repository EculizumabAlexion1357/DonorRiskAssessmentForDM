## SEO / Indexing (IMPORTANT)
This build is indexable by search engines (removed `noindex` tags) and includes:
- meta description + Open Graph tags
- Schema.org structured metadata (JSON-LD)
- robots.txt + sitemap.xml

### Set your site URL
Update the placeholder base URL in these files **before** publishing on a new domain:
- `sitemap.xml`
- `<link rel="canonical"...>` in each HTML file
- `og:url` in each HTML file

Search/replace:
`https://YOUR-DOMAIN-OR-GITHUB-PAGES-URL/`
with your actual URL (examples):
- GitHub Pages: `https://YOUR-USERNAME.github.io/YOUR-REPO/`
- Houston Methodist: `https://www.houstonmethodist.org/YOUR-PATH/`

### Google Search Console (recommended)
1) Add your site in Google Search Console  
2) Submit `sitemap.xml`  
3) Request indexing for the homepage
